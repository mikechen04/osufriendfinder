### tech stack

| piece        | notes        |
| ------------ | ------------ |
| **runtime**  | Node.js      |
| **server**   | Express      |
| **views**    | EJS          |
| **data**     | Firebase RTDB (and related bits in `.env.example`) |

### run locally

```bash
npm install
cp .env.example .env   # fill in secrets
npm run dev
```

Then open `http://localhost:3000` (or whatever `PORT` you set).
